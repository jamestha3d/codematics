Functions

A function is something that takes an input, and returns an output.

Imagine that we want to deliver a pizza. one more example: say we want to knock the door
For example. if we have the number 60. and we want to know the mathematical sine of this number, we could, on a calculator, look for the sin() function, and then give it an input such as 90, and then it would give us an output of say 0.5. This would look like this:
sin(90) = 5
There are several of such functions on the calculator, such as log(), x^y, etc. what they do is take one or more inputs, for example in the case of x ^y, we have to tell the calculator what x is and what y is. and then it gives us x^y. it turns out we can do this in programming as well. we can create functions as a way to group a set of instructions, so that when we provide the function an input. it runs this set of instructions on that input and gives us the expected output. sin() set of instructions for the computer to be able to perform them repeatedly..

For example If we want to tell someone to ring the door bell 5 times. we can say ring the door bell. ring the door bell. ring the door bell. ring the door bell. ring the door bell. of course it is much easier to just tell the person, ring the door bell 5 times. The same thing applies in computing. sometimes we need the computer to perform a task several times. we do not give it this instruction many times over.

we would write something like: do 5 times. { instructions }
