Getting started

Hi there!

Let's talk about programming. Programming is a way to write instructions for the computer to follow in order to carry out a task.

Unlike in human language where humans understand instructions from context, computers are not that smart. You have to be very precise when giving a computer a set of instructions. This is achieved by using logical and mathematical symbols/ concepts.

You can think of programming as a recipe for the computer to follow. i.e a set of step by step instructions for the computer to follow. We will see more about this later

On this Site we will be learning key programming concepts. Since most programming languages share many key similarities, We will be maintaining a general approach to give the reader a wide grasp. However, when we do give particular code examples, our language of choice will be javascript. This is because javascript is one of the easiest languages for beginners, and all you need to test your code is a browser!
